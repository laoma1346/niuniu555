## 🛠️ 遗留问题处理建议

### 问题1: CharacterStateMachine 缺失

**影响**: 
- 角色动画切换可能不协调
- 战斗中的状态判断（能否闪避/能否攻击）可能冲突
- 关卡流程中的角色控制可能出现问题

**解决方案**:
```cpp
// 新建 CharacterStateMachineComponent
UCLASS()
class UCharacterStateMachineComponent : public UActorComponent
{
    // 当前状态
    ECharacterState CurrentState;
    
    // 状态切换
    bool TryChangeState(ECharacterState NewState);
    
    // 状态查询
    bool CanAttack() const { return CurrentState == Idle || CurrentState == Move; }
    bool CanDodge() const { return CurrentState != Hit && CurrentState != Stun; }
};
```

**建议**: 在Phase 6开始前，用1-2天快速实现基础版本。

### 问题2: 随机词条生成

**现状**: 框架已预留（AffixPoolID, MaxAffixCount），但无实际生成逻辑

**解决方案**: 在Phase 7经济系统中一起实现
- 词条库数据资产（AffixLibraryDataAsset）
- 随机生成算法（权重抽取）
- 冲突词条互斥规则

### 问题3: 测试资源整理

**现状**: Content/Weapons/ 中有测试资源（BP_TestEquipment, DA_TestSword_old）

**建议**:
